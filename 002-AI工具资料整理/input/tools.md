# 待整理工具

## Codex (OpenAI)

官网：https://openai.com/index/introducing-codex/

我知道的信息：

- OpenAI 推出的 CLI 编码代理工具，终端里用自然语言指挥 AI 写代码
- 基于 GPT 模型，能读代码库、运行命令、改文件
- 支持多种编程语言
- 目前以 macOS/Linux 终端为主
- 定价：具体价格待核实，一般需 OpenAI API Key
- 特点：强调 agent 自主执行能力，能理解整个项目上下文

## Cursor

官网：https://cursor.com

我知道的信息：

- 基于 VS Code 深度定制的 AI 编辑器，不是插件
- 内置 AI 代码补全、聊天、代码库问答
- 支持多种模型切换（GPT-4、Claude 等）
- 有免费版（Hobby）和 Pro 付费版（$20/月左右，待核实）
- 跨平台：macOS、Windows、Linux
- 特点：编辑器原生 AI 集成，体验比插件更流畅

## Claude Code (Anthropic)

官网：https://docs.anthropic.com/en/docs/claude-code

我知道的信息：

- Anthropic 推出的 CLI 编码工具，终端里和 Claude 对话写代码
- 支持读取项目、编辑文件、执行命令、Git 操作
- 基于 Claude 模型（Sonnet/Opus）
- 跨平台：macOS、Linux、Windows（WSL）
- 定价：需要 Anthropic API Key 或通过 Claude 订阅，具体待核实
- 特点：强调安全性和代码质量，上下文窗口大

## VS Code + GitHub Copilot

官网：https://github.com/features/copilot

我知道的信息：

- VS Code 是微软免费开源的代码编辑器
- GitHub Copilot 是 VS Code 的 AI 扩展插件
- 功能：代码补全、Copilot Chat、代码审查、工作区感知
- 定价：Copilot Free 免费版有限额，Copilot Pro 约 $10/月（待核实），Business/Enterprise 企业方案
- 跨平台：macOS、Windows、Linux（VS Code 全平台）
- 特点：VS Code 生态 + 海量插件，用户基数最大
