# 010 - 月夜狼人杀

单机版狼人杀网页游戏，与 AI 玩家同桌对战。

## 玩法

- 支持 **6 人局** 与 **8 人局**
- 角色：狼人、预言家、女巫、猎人、村民
- 完整流程：夜晚（狼刀 / 查验 / 女巫）→ 白天（讨论 / 投票）→ 猎人开枪
- 胜利条件：狼人全灭则好人胜；狼人数量 ≥ 好人则狼人胜

- 狼人玩家可看到队友标识；预言家有查验记录面板
- 支持加速模式（顶栏按钮，偏好会保存）
- 可点击牌桌选目标；讨论阶段可选择发言

## 启动

**推荐（Mac）：双击 `打开游戏.command`**，会自动启动本地服务并打开浏览器。

或在终端：

```bash
cd 010-狼人杀
./start.sh
```

也可以直接双击 `index.html` 打开（已支持本地文件模式）。

若页面空白或按钮无反应，请 **强制刷新**（Mac：`⌘ + Shift + R`），或使用上面的启动脚本。

## 技术栈

- 纯 HTML / CSS / JavaScript（无构建依赖）
- Retro-Futuristic 视觉：Orbitron + Space Mono、CRT 扫描线、环形牌桌
